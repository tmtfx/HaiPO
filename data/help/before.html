Before running...

I used python version 3.10 and the program needs some modules installed:

  • Haiku-PyAPI: a collection of Python bindings for Haiku APIs

  • polib: python module to handle Portable Object files (along with Portable Object Templates and Machine Object)

  • enchant: python module to handle Portable Object files (along with Portable Object Templates and Machine Object)

  • translate-toolkit: a set of tools for localizers, used to handle the translation memory and to check the po files

  • Levenshtein: a python module used to evaluate which string should be picked from translation memory

  • babel: a set of tools for internationalizing Python applications, used to configure user language.
